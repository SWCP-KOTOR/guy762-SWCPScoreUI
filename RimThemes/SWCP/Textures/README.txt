Here textures used to replace Rimworld's default UI textures, for the complete list of textures available to replace, please download the "Vanilla UI textures pack".

Video background must be in WEBV format (encoded with VP8, it will NOT work with VP9 or AV1) , and his resolution should be 1280x720 to avoid lags.

Since RimThemes NX rev10 you can specify multiple versions of the same texture and Rimworld will pick up a random version when the theme is selected or the game start, simply put a '@' suffix followed with a sequential number to the texture name (ex: Widgets.ButtonBGAtlasMouseover@1.jpg, Widgets.ButtonBGAtlasMouseover@2.jpg)


For example if you want to have 3 random main background image, these 3 background images need to be named as the following :
UI_BackgroundMain.BGPlanet.jpg       : background 1 (for compatibility reason with older RimThemes versions you should always name one texture without the suffix)
UI_BackgroundMain.BGPlanet@1.jpg     : background 2
UI_BackgroundMain.BGPlanet@2.jpg     : background 3

/!\ All filenames are case sensitive !
/!\ If you have at least one texture without the suffix, your theme will still work with older RimThemes releases.
