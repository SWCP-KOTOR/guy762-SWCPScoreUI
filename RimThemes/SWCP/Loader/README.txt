-BGLoader.jpg    : Image used as background loader image
-BGLoader@1.jpg  : [NX rev10 min] Another image used as background loader image 
                   (you can add as many as you want, the next will be BGLoader@2.jpg etc...) 
                   RimThemes will pick randomly one image for the loader every time the game restarts

-Loader.png      : Animated PNG image used as animation during loader screen ( GIF not supported !)
-LoaderBar.png   : Texture used to draw the loader bar
-TextBar.png     : Texture used as background in the loading screen detail zone
-NOLOADER	 : [2019R2 min] Optional, if you create this file (empty without extension), no loading animation are displayed

/!\ All filenames are case sensitive !
/!\ Always put at least one BGLoader.jpg without suffix to maintain compatibility with previous RimThemes versions.
